## Running the Project

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the practice server (located in the `server` folder):
   ```sh
   node server
   ```
   **Note:** Restarting the server resets all data.
3. Run the Angular app:
   ```sh
   ng serve
   ```
4. Open the app in your browser at `http://localhost:4200/`.
